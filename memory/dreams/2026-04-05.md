# 2026-03-26 Memory Note

## 本地 Whisper 转录验证结果

**测试文件**: `/data/workspace/transcript_0315_full.txt` (0315 Meetup 录音)
**对比基准**: PPT 大纲 `openclaw-meetup-speech-outline.md`

### 结论
1. **本地 whisper 能用但中文识别准确率一般** (~60-70%)
2. **用 medium/large 模型可以提升** (当前可能用的是 base/small)
3. **有 API key 后用 OpenAI Whisper 效果更好** (推荐长期方案)

### 错误示例
- "大魔器" → 大模型
- "吹本" → prompt
- "A轩" → AI
- "秘靠" → MiMo

### 可用性
- 能识别大致主题（大模型、小米/MiMo、自动化）
- 技术细节缺失（架构、成本、通信方案）
- 需人工校对才能使用
