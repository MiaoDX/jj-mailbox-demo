## 📋 使用指南

### 添加新任务
1. 在「活跃任务」表格新增一行
2. 任务ID格式：`{类型}-{简述}-{日期}`，如 `blog-openclaw-setup-0325`
3. 状态初始为 `🟡 进行中`

### 完成任务
1. 将任务从「活跃」移到「已完成」
2. 填写完成时间和交付物
3. 立即执行：`git add TASK-BOARD.md && git commit -m "task: 完成任务 XXX"`

### 状态标记
- 🟡 进行中
- ⏸️ 暂停
- ✅ 完成
- ❌ 取消
